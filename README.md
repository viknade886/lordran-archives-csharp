# Lordran Archives

A Dark Souls item archive web application built with ASP.NET Core, Entity Framework, and SQLite.

**Live Frontend:** https://viknade886.github.io/lordran-archives/index.html  
**Backend API:** https://lordran-archives-csharp-production.up.railway.app/swagger

---

## EN — Overview

A full-stack app where users can browse, submit, and manage Dark Souls items.

**Features:**
- JWT authentication with role-based access (user / operator / admin)
- Browse and filter items by category
- Submit items for approval
- Operators and admins can approve or delete items
- 11 NUnit tests with over 70%+ coverage

**Tech Stack:** ASP.NET Core, Entity Framework Core, SQLite, JWT, NUnit

---

## BG — Описание

Full-stack приложение за архивиране на предмети от Dark Souls.

**Функционалности:**
- JWT автентикация с роли (user / operator / admin)
